Урок 1: Основы CSS

CSS (Cascading Style Sheets) - язык стилей для оформления HTML-документов.

Синтаксис:

селектор {
    свойство: значение;
}

Урок 2: Flexbox

Flexbox - современная модель для создания гибких макетов.

.container {
    display: flex;
    justify-content: center;
    align-items: center;
}
Flex элемент

Урок 3: Анимации

CSS-анимации позволяют создавать плавные переходы.

@keyframes example {
    from { background: red; }
    to { background: yellow; }
}

.element {
    animation: example 2s infinite;
}
